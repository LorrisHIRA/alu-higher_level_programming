# JavaScript Objects, Scopes and Closures

## Description
This project focuses on JavaScript **classes**, **objects**, **inheritance**, **closures**, and **scope**.
All scripts are written in JavaScript and executed using **Node.js 14.x** on **Ubuntu 20.04 LTS**.

The goal is to understand how JavaScript handles object-oriented programming,
method inheritance, and function closures while following strict coding standards.
