# JavaScript Warm Up

This repository contains JavaScript scripts that are part of the **JavaScript warm-up exercises** for the ALX Higher Level Programming curriculum. Each script is written in **Node.js**, follows **semistandard coding style**, and is executable on **Ubuntu 20.04 LTS with Node 14.x**.

## Requirements

- **Node.js 14.x**  
  Install with:
  ```bash
  curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
  sudo apt-get install -y nodejs
